# bamboo

R package for exploring and describing panel data

## Overview

This is an R package that I developed for my personal use to make it easier for myself to work with panel data. However, everyone is welcome to use it if they find it helpful. 

## Installation

```
library(devtools)
install_github("dtereshch/bamboo")
```

## Usage
```
library(bamboo)

?decompose_factor
?decompose_numeric
?describe_balance
?describe_dimensions
?describe_incomplete
?describe_patterns
?describe_periods
?make_panel
?plot_heterogeneity
?plot_missing
?plot_patterns
?plot_periods
?production
?summarize_missing
?summarize_numeric
?summarize_transition
```
