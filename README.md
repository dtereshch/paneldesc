# bamboo

Descriptive Analysis and Visualization for Panel Data

## Description

This package provides a comprehensive set of tools for describing and visualizing panel data structures, as well as for summarizing and visualizing variables within a panel data context.

## Installation

```
library(devtools)
install_github("dtereshch/bamboo")
```

## Usage
```
library(bamboo)

?decompose_factor
?decompose_numeric
?describe_balance
?describe_dimensions
?describe_incomplete
?describe_patterns
?describe_periods
?make_panel
?plot_heterogeneity
?plot_missing
?plot_patterns
?plot_periods
?production
?summarize_missing
?summarize_numeric
?summarize_transition
```
