# bamboo

R package for exploring and describing panel data

## Overview

This is an R package that I developed for my personal use to make it easier for myself to work with panel data. However, everyone is welcome to use it if they find it helpful. 

## Installation

```
library(devtools)
install_github("dtereshch/panelDT")
```

## Usage
```
library(panelDT)

?decompose_variation
?describe
?describe_participation
?describe_transition
?explore_incomplete
?find_incomplete
?plot_heterogeneity
?plot_participation
?production
```
